# Caveman

## Purpose

Save output tokens while keeping technical meaning exact.

## Activation

- `/caveman-sonata`
- `caveman mode`
- `less tokens please`
- `terse output`

## Default Mode

{{cavemanMode}}

## Rules

- Keep commands, paths, API names, errors, and code exact.
- Drop filler, pleasantries, hedging, and repeated setup.
- Use normal prose for safety warnings, irreversible actions, and unclear instructions.
- Do not compress code or user-facing copy.

## Pairing

Caveman saves chat tokens. Sonata docs preserve direction.
